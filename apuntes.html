Curso de Introduccion a la Terminal y Linea de Comandos

Clase 1 - ¿Que es la terminal?

La terminal es un programa que ejecuta líneas de comandos, las cuales a su vez realizan acciones en el sistema operativo. Hay dos conceptos clave: la terminal y la shell. La terminal es una interfaz que recibe comandos y los pasa a la shell para su ejecución. La shell es la que realmente lleva a cabo el proceso de ejecutar los comandos, comunicándose con el procesador y la memoria RAM.

A diferencia de la interfaz gráfica convencional, la terminal permite interactuar con el sistema operativo mediante comandos. La interfaz gráfica es un programa adicional que ocupa espacio en la memoria, mientras que la terminal es más ligera y permite comunicarse directamente con los recursos del sistema operativo. Además, en situaciones donde la interfaz gráfica no está disponible o se daña, la terminal sigue siendo una herramienta útil.

Se recomienda utilizar la terminal en sistemas operativos basados en Linux o tener instalado el Windows Subsystem for Linux en Windows para seguir el curso.

Clase 2 - Instalar WSL - usa Linux dentro de Windows

En esta clase aprenderás a instalar el WSL (Windows Subsystem for Linux), una herramienta esencial para desarrolladores de software que utilizan Windows 10 o Windows 11. El WSL permite ejecutar una línea de comandos de Linux desde Windows, facilitando el desarrollo en un entorno Linux.

Si ya tienes una terminal basada en Unix instalada o utilizas Linux nativamente o macOS, puedes omitir esta clase. Sin embargo, si estás en Windows 10 u 11 y aún no tienes instalado WSL con algún sistema Linux, sigue las siguientes instrucciones detalladas:

1. Abre Windows PowerShell como administrador desde el menú de inicio.
2. Ejecuta el comando "wsl --install" para instalar el WSL. Este proceso permitirá ejecutar distribuciones de Linux como Ubuntu o Debian en tu computadora.
3. Espera a que la instalación llegue al 100% y presiona Enter nuevamente para iniciar la instalación del sistema operativo Ubuntu.
4. Reinicia tu computadora después de completar la instalación.
5. Configura Ubuntu proporcionando un nombre de usuario y contraseña.
6. Una vez reiniciada la computadora, accede a Ubuntu desde la aplicación Terminal en Windows.
7. Listo, ahora tienes WSL y Ubuntu instalados en tu máquina, lo que te permite utilizar una terminal de Linux directamente desde Windows para tus tareas de desarrollo.
