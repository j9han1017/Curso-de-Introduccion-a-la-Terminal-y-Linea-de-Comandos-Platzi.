Curso de Introduccion a la Terminal y Linea de Comandos

Clase 1 - ¿Que es la terminal?

La terminal es un programa que ejecuta líneas de comandos, las cuales a su vez realizan acciones en el sistema operativo. Hay dos conceptos clave: la terminal y la shell. La terminal es una interfaz que recibe comandos y los pasa a la shell para su ejecución. La shell es la que realmente lleva a cabo el proceso de ejecutar los comandos, comunicándose con el procesador y la memoria RAM.

A diferencia de la interfaz gráfica convencional, la terminal permite interactuar con el sistema operativo mediante comandos. La interfaz gráfica es un programa adicional que ocupa espacio en la memoria, mientras que la terminal es más ligera y permite comunicarse directamente con los recursos del sistema operativo. Además, en situaciones donde la interfaz gráfica no está disponible o se daña, la terminal sigue siendo una herramienta útil.

Se recomienda utilizar la terminal en sistemas operativos basados en Linux o tener instalado el Windows Subsystem for Linux en Windows para seguir el curso.

Clase 2 - Instalar WSL - usa Linux dentro de Windows

En esta clase aprenderás a instalar el WSL (Windows Subsystem for Linux), una herramienta esencial para desarrolladores de software que utilizan Windows 10 o Windows 11. El WSL permite ejecutar una línea de comandos de Linux desde Windows, facilitando el desarrollo en un entorno Linux.

Si ya tienes una terminal basada en Unix instalada o utilizas Linux nativamente o macOS, puedes omitir esta clase. Sin embargo, si estás en Windows 10 u 11 y aún no tienes instalado WSL con algún sistema Linux, sigue las siguientes instrucciones detalladas:

1. Abre Windows PowerShell como administrador desde el menú de inicio.
2. Ejecuta el comando "wsl --install" para instalar el WSL. Este proceso permitirá ejecutar distribuciones de Linux como Ubuntu o Debian en tu computadora.
3. Espera a que la instalación llegue al 100% y presiona Enter nuevamente para iniciar la instalación del sistema operativo Ubuntu.
4. Reinicia tu computadora después de completar la instalación.
5. Configura Ubuntu proporcionando un nombre de usuario y contraseña.
6. Una vez reiniciada la computadora, accede a Ubuntu desde la aplicación Terminal en Windows.
7. Listo, ahora tienes WSL y Ubuntu instalados en tu máquina, lo que te permite utilizar una terminal de Linux directamente desde Windows para tus tareas de desarrollo.

Clase 3 - Aprendiendo a caminar en la terminal

En los sistemas operativos, la organización de archivos se realiza en una estructura de carpetas en forma de árbol jerárquico. En Linux, la carpeta raíz "/" contiene carpetas como "Home," que a su vez contiene carpetas para cada usuario.

Al abrir la terminal en Linux, se muestra información como el nombre de usuario, el nombre de la computadora, y la ubicación actual (~ indica la carpeta actual). El comando "pwd" muestra el directorio actual.

El comando "cd" (Change Directory) se utiliza para moverse entre carpetas en la terminal. Se pueden usar atajos como "~" para el directorio home, ".." para retroceder un nivel, y "/" para ir a la raíz.

El comando "ls" (List) muestra los archivos y carpetas en la ubicación actual. Con la opción "-l," se obtiene una lista detallada que incluye información como permisos, tamaño y fecha de modificación. La opción "-h" hace que los tamaños sean legibles para humanos.

El comando "file" proporciona información sobre el tipo de elemento seleccionado en una carpeta, como si es un directorio o un enlace simbólico.

Clase 4 - Manipulando archivos y directorios

Después de aprender a navegar y listar contenidos en la terminal, es crucial aprender a manipular archivos con comandos básicos en Linux.

1. Crear un directorio (mkdir):
- El comando "mkdir" (Make Directory) se utiliza para crear un directorio. Simplemente escribe el nombre del directorio que deseas crear, y puedes crear varios directorios al mismo tiempo.

mkdir DirectorioInteresante

2. Crear archivos (touch):
- El comando "touch" se utiliza para crear archivos. Simplemente escribe el nombre de los archivos que deseas generar delante del comando.

touch Secreto1 Secreto2 SecretoSecretario

3. Copiar archivos (cp):
- El comando "cp" (Copy) se utiliza para copiar archivos. Especifica el nombre del archivo que deseas copiar y el nombre con el que quieres que se guarde la copia.

cp Secreto1 Secreto_bk

4. Mover un directorio (mv):
- El comando "mv" (Move) se utiliza para mover archivos o directorios. Especifica el nombre del archivo que deseas mover y la ubicación a la que se moverá.

mv SecretoQueSeHizoPublico ../SecretosPublicos/

5. Eliminar archivos (rm):
- El comando "rm" (Remove) se utiliza para eliminar archivos o carpetas. Simplemente escribe el nombre o la ruta del elemento que deseas eliminar.

rm SecretoQueNadieDebeVer

- Opciones útiles de "rm":
- `-i` (interactive): Pregunta si estás seguro de eliminar el archivo.
- `-r` (recursive): Elimina todo lo que está dentro de una carpeta.
- `-f` (force): Fuerza la eliminación sin preguntar.


rm -i SecretoConfidencial
rm -r CarpetaConArchivos
rm -f ArchivoEnUso



Comandos mkdir [nombre_directorio]: crea un directorio también puedes añadir la ruta

touch: crea un archivo, si no indicas la extension por defecto sera .txt

cp [archivo]
[ruta]: nos permite duplicar archivos, para duplicar directorios con su contenido añadir el modificador -r que indica recursividad

mv [archivo]
[ruta] : mover un archivo hacia un directorio. También se usa para renombrar archivos mv [archivo] [nuevo nombre] no olvidar la extension del archivo Para mover directorios añadir el modificador -r

rm: elimina archivos, con -i indica un mensaje de confirmación en consola para tener mas control sobre la elección de que archivos queremos eliminar. Para eliminar un directorio añadir -r y -ri para usar el mensaje de confirmación

Clase 5 - Explorando el contenido de nuestros archivos

1. Comandos Head y Tail:
- Head imprime las primeras 10 líneas de un archivo, y tail imprime las últimas 10 líneas.
- Sintaxis simple: `head archivo.txt` o `tail archivo.txt`.
- Opción -n para especificar el número de líneas, por ejemplo: `head proyecto.html -n 20`.

2. Comando Less:
- Permite ver todo el contenido de un archivo y navegar por él.
- Sintaxis: `less [nombreDelArchivoParaAbrir]`.
- Interfaz gráfica para inspeccionar el documento.
- Uso de "/" para buscar palabras dentro del documento.
- Para salir de la interfaz, presiona "q".

3. Comando xdg-open:
- Abre un archivo en el programa predeterminado para su extensión.
- Ejemplo: `xdg-open archivo.html`.
- Puede requerir instalación con `sudo apt install xdg-utils` si no está disponible.
- Se utiliza para abrir archivos y, al ejecutarse para un archivo `.html`, muestra una interfaz similar a less.

4. Abrir una Carpeta con el Comando Nautilus:
- El comando `nautilus` abre una carpeta utilizando el programa Nautilus.
- Sintaxis: `nautilus carpeta`.
- Similar a xdg-open pero específicamente para carpetas.

5. Manejo de Errores:
- Si xdg-open no está instalado, seguir las instrucciones del mensaje de error para instalarlo.
- Ejemplo de instalación: `sudo apt install xdg-utils`.
En resumen, estos comandos permiten inspeccionar rápidamente el contenido de archivos, visualizar todo el documento de manera interactiva, abrir archivos en programas predeterminados y explorar carpetas desde la terminal.
